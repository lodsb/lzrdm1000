JJack Overview

JJack is a JACK audio driver for the Java Sound API and a framework for the Java programming language that allows Java applications to use the JACK Audio Connection Kit.

  • JACK-driver for the standard Java Sound API via javax.sound.sampled.spi.MixerProvider
  • Use Java to create audio processors (effects processors, synthesizers etc.).
  • JavaBeans-compatible event mechanism to interconnect audio processors.
  • Native bridge code uses java.nio.ByteBuffer for efficient direct memory access from Java.
